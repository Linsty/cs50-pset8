The requirements for pset8 Homepage can be found on the website of CS50 2021. (See links & credits).
I decided to create a little framework where index.html is the frame and the other files will be integrated in this page by using an iframe.
At the top of index.html I created a navigation bar with the links to all the created pages.
The target of these links were set to the iframe and will be shown this way in index.html.
However you will see just 1 page, there are at least 4 pages on the webpage and this way I met the first rule of the assignment.

HTML

The second rule was to use at least 10 distinct HTML tags besides <html>, <head>, <body> and <title>. This wasn’t a hard thing to do, because I needed already at least 10 different tags for my index.html file.
First of all you need a <meta>, <link> and <script> tag to use Bootstap, an external stylesheet and an external js fie. I thought it was a good idea to divide the framework in different sections. I used the <section> and the <footer> tag for this.
I love jokes and because I’m Dutch I decided to integrate the Dutch flag and some orange flags as decoration in the header and the footer.I used some <div> tags to make this possible. I still have an empty page and I already have 6 tags!
A good page needs a navigation bar. To create this, I needed to make an unsorted list <ul> with list items <li> of links <a> to the other html files. Since I want them in my page I needed to create an iframe <iframe> where these html files were displayed. And there are the 10 different HTML tags!

Bootstrap

For the navigation bar I used Bootstrap’s navbar and the toggler to hide stuff in a button to make the webpage more responsive.

CSS

To style index.html I created a css stylesheet with the name styles.css and the Harvard staff wanted us to style at least 5 css selectors and use at least 5 different css.properties.
All the items that had a default style I used the body tag to style them. This way I didn’t have to repeat myself all the time.
body{
font-family: 'Noto Sans JP', sans-serif;
font-weight: 500;
text-align: center;
margin-bottom: 0px;
background-color: white;
}
 And wow in just one selector I showed you 5 different CSS properties!

For the colors of the flag I used some class-names like red-space, white-space and blue-space.
I think it’s clear what red-space, white-space and blue-space styled for the website.
For a few elements I preferred an id, like #navigation-bar. I needed that because I wanted to modify the navigation bar to my personal needs.

Javascript

On the website I created a greeter that greets the visitor in the right local timezone.

Responsive webpage

I checked so far if everything looks fine on different sizes. To my opnion it looks good.
